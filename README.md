# 🌐 Projeto Front-End

Este projeto foi desenvolvido utilizando **HTML5**, **CSS3** e **JavaScript**, com foco no aprendizado dos fundamentos do desenvolvimento Front-End e na prática da construção de interfaces e funcionalidades.

## 🚧 Status

> **Atenção:** Este projeto foi desenvolvido **apenas para desktop** e **não possui responsividade**.

O objetivo foi praticar HTML, CSS e JavaScript, sem a implementação de técnicas de adaptação para dispositivos móveis.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)

## 📚 Objetivos

- Praticar estruturação de páginas com HTML.
- Desenvolver layouts utilizando CSS.
- Manipular elementos com JavaScript.
- Aprimorar a lógica de programação.
- Organizar o código seguindo boas práticas.

## 🔄 Melhorias Futuras

- Adicionar responsividade (Mobile First).
- Melhorar a acessibilidade.
- Otimizar o desempenho.
- Refatorar o código.

---

**Projeto desenvolvido para fins de estudo e evolução como desenvolvedor Front-End.**
